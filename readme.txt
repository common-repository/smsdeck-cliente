== Description ==

Plugin de integração do SMSdeck para WordPress. Com este plugin você pode enviar SMS para seus clientes informando status de compras e faturas ou entrar em contato.

**Para lojas virtuais baseadas em WordPress**  
Notificar seus clientes sobre status de compras (ou entregas) via SMS dará a sua loja maior credibilidade, simplesmente pelo fato de que o comprador pode acompanhar o rastreamento da entrega através de mensagens SMS.

Sua loja também pode utilizar o poder do SMS para notificar promoções e datas sazonais.

Para sites, blogs e lojas virtuais  
Um grande problema do email marketing é a caixa de spam, não dá para garantir que o email realmente chegue na caixa de entrada, isto é um fato. O SMSdeck também permite realizar campanhas de SMS marketing.

Toda mensagem enviada vai direto para o celular de destino, praticamente todo mundo tem pelo menos 1 celular e isto abre caminhos para novas oportunidades para divulgar promoções, notificar datas comemorativas e claro, desejar feliz aniversário ao seu cliente, afinal, por traz das compras existem pessoas que adoram ser paparicadas.

== Installation ==

1. Faça upload do plugin "smsdeck-cliente.zip" para o diretório "/wp-content/plugins/" 

1. Ative o plugin na página "Plugins"

1. Acesse o menu SMSdeck -> Configurações

1.2 Acesse <a href="http://smsdeck.com/">SMSdeck</a> e crie sua conta

1.3 Gere uma <a href="http://smsdeck.com.br/documentacao/configuracao/gerar-token-de-acesso.html">chave de acesso API</a>

1. Configure o plugin com as informações de sua conta SMSdeck

== Frequently Asked Questions ==

Enviei alguns torpedos SMS e agora não consigo mais enviar, o que houve?

Quando você cria uma conta no SMSdeck, você recebe alguns créditos para testar o plugin, conforme você envia torpedos SMS os créditos de sua conta SMSdeck são consumidos, quando eles acabam você não consegue enviar novos torpedos até adquirir créditos.

Instalei o plugin no meu site, mas quando tento enviar torpedos SMS aparece uma mensagem "Token inválido", o que é isso?

Esta mensagem aparece em 2 casos:

1.  Você não informou a chave token de acesso a API
2.  Você gerou uma nova chave token e não a inseriu no painel do plugin SMSdeck

Em ambos os casos basta acessar sua conta no SMSdeck, copiar sua chave de acesso e informa-la no plugin SMSdeck Cliente.
